Arzoo Bakery

This project was created for a fictional bakery called Arzoo Bakery using HTML, CSS, and JS. It's designed to work smoothly on various devices, with a clean layout reflecting the bakery's sweet aesthetic. JavaScript adds a mobile-friendly hamburger menu for easy navigation. Check it out to see how I brought this bakery to life online!

Dice Game

The Dice Game is a straightforward project that simulates a dice throw for two players. It displays the winner on the screen, and with a page refresh, you can initiate a new round. Enjoy a quick and fun game of chance!

Drums Kit

This interactive project, created using HTML, CSS and Javascript, allows users to play drums digitally. On mobile devices, tap on-screen buttons to produce beats, while desktop and laptop users can engage with the kit using keyboard buttons or mouse clicks.
